# Phasmophobia RPG v1.1.3

A companion RPG progression tracker for Phasmophobia.

## GitHub Pages

Upload these files to your repository root:

- index.html
- styles.css
- app.js
- data.js
- classes.js
- talents.js
- progression.js
- save.js
- shop.js
- loot.js
- contract.js
- ui.js

Then go to GitHub:

Settings → Pages → Deploy from branch → main → /root

Your app should run at:

https://<your-username>.github.io/<repo-name>/

## Notes

Saves are stored in browser localStorage and can also be exported/imported as JSON.
